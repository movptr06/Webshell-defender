# wsdect
Free webshell detection solution.

Install
```
curl https://raw.githubusercontent.com/movptr06/wsdect/main/setup.sh 2>/dev/null | sudo sh
```
Get update
```
./wsdect update
```
Scan webshell
```
./wsdect scan <PATH>
```

## Used rules
Some rule files were imported from the outside.
